# 💫 Acerca de mi:
👋 ¡Hola! Soy Jheremy William, programador junior de 22 años apasionado por la tecnología y el desarrollo de software, con sede en Perú 🇵🇪. Soy egresado de Computación e Informática y actualmente continúo mis estudios en Ingeniería de Sistemas. Aunque estoy en los primeros pasos de mi carrera, tengo una gran determinación por aprender, evolucionar y aportar valor en cada proyecto en el que participo.<br><br>Me encanta enfrentar desafíos, explorar nuevas tecnologías y construir aplicaciones full-stack de principio a fin. Creo firmemente que el aprendizaje es un proceso constante y disfruto cada oportunidad de mejorar mis habilidades tanto técnicas como blandas.<br><br>🚀 Mis fortalezas:<br>✔️ Rápida capacidad de aprendizaje y adaptación a nuevas tecnologías<br>✔️ Autonomía para llevar un proyecto desde la idea hasta el despliegue<br>✔️ Compromiso con la calidad, los detalles y las buenas prácticas<br>✔️ Enfoque en la mejora continua y el crecimiento profesional<br><br>🌱 Actualmente estoy profundizando en:<br><br>Vue 3 y Vite para construir interfaces modernas y reactivas<br>Node.js y Express para desarrollar backends robustos<br>PostgreSQL (incluyendo PostGIS para datos geoespaciales) como base de datos principal<br>Prisma y Sequelize como ORMs para trabajar con esas bases de datos<br>Estoy en constante aprendizaje y siempre buscando nuevas oportunidades para aplicar mis conocimientos en proyectos reales 🚀💡
